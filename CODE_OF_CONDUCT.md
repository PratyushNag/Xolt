# Code of Conduct

## Our Standard

Participants in the Xolt community are expected to be respectful, direct, and constructive.

## Unacceptable Behavior

- Harassment or intimidation
- Personal attacks
- Publishing private information without consent
- Disruptive or abusive conduct

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

## Reporting

Report conduct issues to the maintainers through the contact path in [SECURITY.md](SECURITY.md).
